# Bug Report – AutomationExercise

## BUG_001 – Login button allows submission with empty fields

- Module: Login
- Severity: High
- Priority: High

### Steps to Reproduce:
1. Open login page
2. Leave email and password empty
3. Click Login button

### Expected Result:
User should see validation message and login should be blocked

### Actual Result:
Login button allows submission without validation
